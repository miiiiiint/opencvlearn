#  shape
##### 使用shape函数会返回一个数组
前两项 shape[0] 和 shape[1] 表示图像的宽度和高度。
Shape[2] 代表通道数。
3 表示图像具有红绿蓝 (RGB) 3个通道。
```python
    shape[0]---宽
    shape[1]---高
    shape[2]---通道数`
```
